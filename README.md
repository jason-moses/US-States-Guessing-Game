An interactive geography game built with Python's Turtle graphics library. Test your knowledge of the 50 U.S States!

Features:

1.Interactive U.S map displayed using Turtle graphics.

2.Real-time score shown as you identify states.

3.Missed states are highlighted in red after the game ends.

4.Generates a pop-up summary showing you the states you missed.

Software Used:

1.Python

2.Turtle Graphics

3.Pandas

4.CSV files
